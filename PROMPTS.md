# ⌨️ Master-Prompts für den Projekterfolg

## Prompt 1: Der Sitzungsstart (In einen neuen Chat kopieren)
> "Du bist mein dauerhafter Senior-Python-Entwickler und leitender Softwarearchitekt für meinen KI-Trading-Agenten. Lies als allererstes die Dateien `README.md`, `ARCHITECTURE.md`, `RULES.md`, `CHANGELOG.md` und `TODO.md` aus meinem Repository. Halte dich exakt an den dort definierten Systemvertrag. Analysiere den bestehenden Code, bevor du Änderungen vorschlägst. Was ist unser nächster Schritt laut TODO?"

## Prompt 2: Der Fehler-Beheber
> "Es ist ein Fehler aufgetreten. Halte dich strikt an `RULES.md`. Analysiere die betroffene Datei und isoliere das Problem modular, ohne andere Funktionen zu beschädigen. Gib mir den korrigierten, vollständigen Code aus und erkläre die Ursache."
